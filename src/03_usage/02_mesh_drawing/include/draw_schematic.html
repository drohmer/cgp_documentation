**draw(drawable, environment)**

1) Set **current shader** attached to drawable element ::(.draw_comment) glUseProgram( drawable.shader ) ::
2.a) Send **uniforms** parameters attached to the **drawable** element ::(.draw_comment) glUniform( drawable.[parameters] ) ... on transform and shading values ::
2.b) Send **uniforms** parameters attached to the **environment** ::(.draw_comment) glUniform( environment.[parameters] ) ... ::
3) Set texture of the drawable element ::(.draw_comment) glBindTexture(GL_TEXTURE_2D, drawable.texture) ::
4) **Draw** calls on the drawable vbo (data already stored on the GPU) ::(.draw_comment) glBindVertexArray( drawable.vao ) :: ::(.draw_comment) glBindBuffer(GL_ELEMENT_ARRAY_BUFFER, drawable.connectivity) :: ::(.draw_comment) glDrawElements(GL_TRIANGLES) ::
5) Cleaning buffers state ::(.draw_comment) glBindVertexArray(0) :: ::(.draw_comment) glBindTexture(GL_TEXTURE_2D, 0) ::
